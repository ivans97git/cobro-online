# 💰 Sistema de Cobro Online

Aplicación web para gestión de cobros con 6 cobradores, desarrollada con Node.js, Express y SQLite.

## 🚀 Características

- ✅ Autenticación segura con JWT
- ✅ 6 usuarios cobradores preconfigurados
- ✅ Registro de cobros (cliente, monto, método de pago)
- ✅ Estadísticas en tiempo real
- ✅ Historial de cobros por cobrador
- ✅ Base de datos SQLite (sin necesidad de instalar MySQL)
- ✅ Interfaz responsive y moderna

## 📋 Requisitos Previos

- Node.js (v14 o superior)
- npm (v6 o superior)

## 🔧 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/cobro-online.git
cd cobro-online

# Instalar dependencias
npm install

# Iniciar el servidor
npm start
